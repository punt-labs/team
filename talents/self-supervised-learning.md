# Self-Supervised Learning

Learning representations without explicit labels.

- Pretext tasks: masked-prediction, contrastive, predictive coding
- Contrastive: SimCLR, MoCo, CLIP — augmentation as the supervision signal
- Joint-embedding architectures and the Yann LeCun energy-based view
- Foundation models as the deployment artifact of self-supervised pretraining
- Evaluation: linear probes, downstream task performance, OOD generalization

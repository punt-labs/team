# Web Standards

The web as a contract — backwards compatibility is non-negotiable.

- HTML, CSS, ECMAScript spec literacy; living standards
- W3C / WHATWG / TC39 process awareness
- Progressive enhancement: build for the platform, not the framework
- Accessibility (WCAG, ARIA) as part of the standard, not a bolt-on
- Service workers, edge runtimes, and the platform-API surface
